MERN → Engineer — Learning Tree (HTML & CSS focused)

A single-page roadmap that turns your MERN knowledge into production-ready engineering skills. Click a node to expand resources.

1. HTML & CSS Fundamentals
Solid web foundations — required before anything else.
  • MDN — HTML Reference & Tutorials — authoritative reference and guides.
  • MDN — CSS Tutorials — from basics to advanced techniques.
  • freeCodeCamp — Responsive Web Design — interactive exercises & projects.
2. Deepen MERN (beyond CRUD)
Add production features to your apps.
  • React — Official Docs — component patterns & hooks.
  • Express — Official Docs — API design and middleware.
  • Node.js — Docs — runtime & core APIs.
  • Socket.IO — Real-time — websockets & live features.
  • Stripe — Payments — payments integration guide.
  • Multer — File uploads — express file handling.
3. Backend Engineering
Design robust, scalable servers and databases.
  • MongoDB Docs — indexing, aggregations, performance.
  • PostgreSQL Docs — when to use SQL & relational design.
  • TypeScript — Official Docs — bring types to JS projects.
  • Jest — Testing — unit & integration testing.
4. System Design
Learn to architect systems that scale.
  • System Design Primer — GitHub — a practical guide to designing large-scale systems.
  • AWS — Getting Started — cloud building blocks for system design.
5. DevOps & Cloud
Shipping, monitoring and running production apps.
  • Docker — Get Started — containerize your apps.
  • AWS Docs — compute, storage, serverless.
  • GitHub Actions — CI/CD — automate builds & deployments.
  • Prometheus — Monitoring — metrics & alerting.
6. Expand Your Toolbox
Other stacks, languages & useful frameworks.
  • Next.js — Fullstack React
  • NestJS — Opinionated Node framework
  • TypeScript — again (great for large codebases)
  • Python — FastAPI / Django
7. Data Structures & Algorithms
Solve problems efficiently and prepare for interviews.
  • LeetCode — Practice problems
  • VisuAlgo — Visualize algorithms & DS
8. Contribute & Build
Real projects and open source accelerate growth.
  • GitHub — Find OSS projects to contribute
  • Dev.to — Write technical posts
Tip: use this page as a checklist — complete one node, build a project that uses it, then move to the next.
